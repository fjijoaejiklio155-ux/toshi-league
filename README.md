# TOSHIリーグ 最小プロトタイプ

投資クイズを題材にした、ネタ寄りのブラウザゲーム試作です。

## ファイル構成

- `index.html` : 画面構造
- `style.css` : 見た目
- `main.js` : ゲームロジック

## 動かし方

### ローカルで確認
`index.html` をブラウザで開くだけで動きます。

### GitHub Pages で公開
1. この3ファイルをリポジトリ直下に置く
2. GitHub の Settings → Pages を開く
3. Branch を `main`、Folder を `/ (root)` に設定
4. 保存すると公開URLが発行されます

## 現在の仕様

- 5問構成
- 1問20秒
- 10択から5つ選ぶ
- 部分点制
- 問題間でランダムに次問2倍イベント
- 架空の相手チームとスコア勝負
